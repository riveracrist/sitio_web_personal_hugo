## ⚙️ Ejecutar localmente

Si deseas clonar y visualizar el sitio en tu máquina:

```bash
# 1. Clona el repositorio
git clone https://github.com/riveracrist/tu-repositorio.git
cd tu-repositorio

# 2. Inicia el servidor de desarrollo
hugo server -D
```

Luego abre tu navegador en `http://localhost:1313`

> **Requisito:** Tener [Hugo instalado](https://gohugo.io/installation/) (versión extendida recomendada).
